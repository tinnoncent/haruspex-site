# HARUSPEX

## Trading intelligence, made legible.

HARUSPEX is a local-first cryptocurrency research, simulation, risk-management, and Kraken Spot execution platform. It is designed for users who want to test a defined process, inspect the system's state, and retain control over when—or whether—live execution is enabled.

**Simulation first. Transparent risk controls. Local credentials. No hidden telemetry.**

---

## What HARUSPEX does

- Runs synthetic simulation for safe initial validation
- Uses real Kraken market data in paper mode
- Supports deliberately gated Kraken Spot execution
- Displays engine health, data freshness, exposure, drawdown, and circuit-breaker state
- Separates strategy decisions from the independent risk layer
- Provides backtesting and walk-forward analysis
- Visualizes observed event outcomes through the Probability Lattice
- Stores configuration and API credentials locally

HARUSPEX does not guarantee profitable trades and should not be treated as investment advice.

## Operating progression

### 1. Simulation
Learn the interface and verify the engine without connecting an exchange account.

### 2. Paper trading
Use real Kraken market data while simulated execution records system behavior without submitting live orders.

### 3. Live execution
Live mode remains disabled until the required configuration, credential validation, safety checks, and user acknowledgements are complete.

## Built around risk visibility

HARUSPEX surfaces the information an operator needs before allowing a system to act:

- Current operating mode
- Exchange and data connection status
- Heartbeat and data age
- Equity and current exposure
- Daily profit and loss
- Maximum drawdown
- Circuit-breaker status
- Directional model conviction
- Observed outcome statistics
