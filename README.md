# HARUSPEX Website V2

This version is intentionally self-contained.

## Deploy
Upload or replace only `index.html` in the root of your GitHub repository. Vercel will redeploy automatically.

## Before launch
Search inside `index.html` for:

- `const LINKS=`
- `support@example.com`
- `sales@example.com`

Replace those placeholders with the final Gumroad and support links.

Both the HARUSPEX sigil and dashboard preview are embedded directly inside the HTML, so there are no external image paths to break.
